Trabalhando com Listas

Listas Ordenadas

As listas ordenadas são listas numeradas - para cria-las use a tag OL

  1. Acordar
  2. Tomar café
  3. Banhar
  4. Ir para a escola

A tag LI representa o item

na TAG OL pode ser utilizado o parametro type para alterar o tipo da numeração e o parametro start para indicar a posição que a lista vai começar

Listas não ordenadas

Listas que não segue uma ordem - para cria-las use a tag UL

  • Essa
  • é
  • uma
  • não ordenada

A tag UL também pode ter seu tipo alterado

Juntandos listas

É possivel misturar e criar lista ordenas e nao ordenadas

  1. Filmes
    • Um amor para recordar
    • As branquelas
    • Brilho eterno de uma mente sem lembraças
  2. Séries
    • CSI - qualquer um
    • Mentalist
    • 24 horas
